# Arambhik

## Overview

Arambhik is an educational website developed as an AI-assisted no-code web project. It was built to provide an engaging and modern online presence while exploring AI-powered website development.

## Built With

- Bolt.new
- ChatGPT
- React
- TypeScript
- Vercel (Deployment)

## Features

- Responsive design
- Modern landing pages
- Mobile-friendly interface
- Fast loading experience
- Contact and information pages
- Clean and intuitive navigation
